## 🚀 Features

- 🔒 **Seamless Authentication** — Secure sign-in for users and recruiters.
- 🗂️ **Resume Upload & Storage** — Fast and easy upload and management.
- 🤖 **AI-Powered Evaluation** — Analyze resumes and match candidates to jobs using advanced AI.
- 🎯 **Job Matching** — Smart matching between resumes and job requirements.
- 💎 **Clean, Reusable UI** — Modern and responsive user interface.

---

## 🌟 Importance

In today's competitive job market, finding the right candidate is challenging. **AI Resume Analyzer** streamlines the process by leveraging the latest AI technologies to:
- Save HR teams countless hours in manual screening.
- Ensure fair, unbiased, and skills-focused shortlisting.
- Provide candidates with actionable feedback.

## 📦 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ABHAY627/AI-RESUME-ANALYZER.git
   cd AI-RESUME-ANALYZER
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the app:**
   ```bash
   npm start
   ```
   The app will be available at `http://localhost:3000`.
